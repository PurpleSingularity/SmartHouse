# Mission

## What

A reusable configuration scaffold that provides standardized AI agent workflows, coding skills, and documentation pipelines for projects using Claude Code and OpenAI Codex.

## Who

Development teams integrating Claude Code (Anthropic CLI) or Codex into their projects who want pre-built agents, quality standards, and automated documentation generation.

## Why

Setting up AI-assisted development workflows from scratch requires defining agent behaviors, coding standards, permission models, and prompt engineering patterns for each project. This template provides a tested, copy-in-place foundation.
